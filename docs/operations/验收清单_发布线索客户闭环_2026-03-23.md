# 发布-线索-客户闭环验收清单（可打勾）

更新时间：2026-03-23
适用版本：发布任务中心 + 线索池 + 客户高亮联动

## 0. 验收前准备

- [ ] 后端服务已启动（含数据库）
- [ ] 前端桌面端已启动并可登录
- [ ] Alembic 版本为 head（`20260323_01`）
- [ ] 当前账号具备发布、线索、客户页面访问权限

## 1. 场景A：发布任务查看链路 -> 线索定位

### 步骤

- [ ] 打开发布任务页，创建一条新任务（必填：平台、账号、标题、内容）
- [ ] 在任务行点击“领取”
- [ ] 在任务行点击“回填”，录入至少 `leads >= 1`，提交
- [ ] 在同一任务行点击“查看链路”

### 预期结果

- [ ] 成功跳转到线索池页面
- [ ] 页面顶部出现“已从任务 #X 跳转定位线索 #Y”提示
- [ ] 线索表格中目标线索行高亮显示

## 2. 场景B：线索转客户（人工确认）

### 步骤

- [ ] 在线索池找到高亮线索，点击“转客户”
- [ ] 在弹窗中输入昵称（可留空）并确认

### 预期结果

- [ ] 页面提示“线索 #Y 已转客户 #Z”
- [ ] 当前线索行“关联客户”列显示 `#Z`
- [ ] “转客户”按钮变为禁用并显示“已转客户”
- [ ] 出现“查看客户”按钮

## 3. 场景C：客户页高亮定位

### 步骤

- [ ] 在线索页点击“查看客户”（或转客户后自动跳转）

### 预期结果

- [ ] 成功跳转到客户管理页
- [ ] 页面顶部出现“已从线索 #Y 跳转，定位客户 #Z”提示
- [ ] 客户列表中目标客户行高亮显示

## 4. 异常场景

### 4.1 发布任务无链路

- [ ] 在未回填/无线索数据任务点击“查看链路”
- [ ] 提示“暂无关联线索，请先回填发布结果”

### 4.2 重复转客户保护

- [ ] 对已有关联客户的线索重复尝试转客户
- [ ] 前端按钮禁用，不能重复触发

### 4.3 权限保护

- [ ] 使用无权限账号访问他人线索转客户接口
- [ ] 返回 403/无权限提示

## 5. 接口验收（抽样）

- [ ] `GET /api/publish/tasks/{task_id}/trace` 返回 `lead_id/customer_id` 字段
- [ ] `GET /api/lead/list` 返回 `customer_id` 字段
- [ ] `POST /api/lead/{lead_id}/convert-customer` 首次创建客户成功
- [ ] 再次调用转客户接口返回同一客户（幂等）

## 6. 回归检查

- [ ] 发布任务基础流程（创建/领取/回填/驳回/关闭）无回归
- [ ] 线索状态流转与归我操作无回归
- [ ] 客户新增录入流程无回归
