# 完成验证

## 触发条件
- 即将告诉用户"完成"
- 准备提交代码

## 验证清单

### 后端改动
- [ ] 服务启动无报错
- [ ] 新接口可调用（Swagger测试）
- [ ] 返回数据格式正确

### 前端改动
- [ ] 页面能正常打开
- [ ] 功能可点击操作
- [ ] 无控制台报错

### 数据库改动
- [ ] 迁移脚本已生成
- [ ] 旧数据不受影响

### AI相关
- [ ] prompt能正常工作
- [ ] 返回格式符合预期

## 快速测试命令

```bash
# 后端启动
cd backend && uvicorn main:app --reload

# 前端启动
cd desktop && npm run dev

# 测试接口
curl http://localhost:8000/docs
```

## 未验证不说完成
如果没有实际运行验证，告诉用户"已修改，请验证"而非"已完成"。
