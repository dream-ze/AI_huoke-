## 🚀 快速启动指南

### 方式 1：Docker（推荐，5 秒启动）
```bash
cd backend
docker-compose up -d
```

**自动启动：**
- ✅ PostgreSQL 数据库
- ✅ FastAPI 后端
- ✅ Ollama AI 模型

访问：http://localhost:8000

### 方式 2：本地开发
```bash
cd backend

# 安装依赖
pip install poetry
poetry install

# 初始化数据库
python init_db.py

# 启动服务
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## 🐛 常见问题解决

| 问题 | 解决方案 |
|------|---------|
| 数据库连接失败 | 检查 DATABASE_URL，确保 PostgreSQL 运行 |
| Token 过期 | 重新登录获取新 token |
| CORS 错误 | 修改 .env 中的 CORS_ORIGINS |
| Ollama 连接失败 | 确保 Ollama 已启动，检查 OLLAMA_BASE_URL |
| 端口被占用 | 修改 docker-compose.yml 中的端口号 |
