## 🐛 常见问题解决

| 问题 | 解决方案 |
|------|---------|
| 数据库连接失败 | 检查 DATABASE_URL，确保 PostgreSQL 运行 |
| Token 过期 | 重新登录获取新 token |

| Ollama 连接失败 | 确保 Ollama 已启动，检查 OLLAMA_BASE_URL |
