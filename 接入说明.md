## AI_huoke 调用方式

在 AI_huoke 后端任意位置引入以下函数即可：

```python
import requests

def collect_content(
  platform: str,
  keyword: str,
  max_items: int = 10,
  export_sample_excel: bool = False,
  sample_size: int = 20,
) -> dict:
    """
    调用采集服务，返回采集结果。
    platform: 目前支持 "xiaohongshu"
    keyword:  搜索关键词，如 "贷款"
    max_items: 最多采集条数
    export_sample_excel: 是否额外导出Excel样例
    sample_size: Excel样例导出条数
    """
    resp = requests.post(
        "http://127.0.0.1:8005/api/collect/run",
        json={
            "platform": platform,
            "keyword": keyword,
        "max_items": max_items,
        "export_sample_excel": export_sample_excel,
        "sample_size": sample_size,
        },
        timeout=180
    )
    resp.raise_for_status()
    return resp.json()
```

### 调用示例

```python
result = collect_content(
  platform="xiaohongshu",
  keyword="贷款",
  max_items=10,
  export_sample_excel=True,
  sample_size=10,
)
print(result["count"])       # 实际采集条数
print(result["task_status"]) # finished / partial / failed
print(result["stats"])       # 扫描/解析/去重/失败统计
print(result["items"])       # 列表，每条含 title/author_name/snippet/url/like_count 等字段
print(result.get("sample_excel_file"))  # Excel样例路径（未开启则为None）
```

详情补采示例：

```python
detail_resp = requests.post(
    "http://127.0.0.1:8005/api/collect/detail",
    json={"platform": "xiaohongshu", "url": "https://www.xiaohongshu.com/explore/xxxx"},
    timeout=180,
)
detail_resp.raise_for_status()
print(detail_resp.json())
```

## 服务管理（SSH 到服务器执行）

```bash
# 查看服务状态
systemctl status browser_collector

# 查看实时日志
journalctl -u browser_collector -f

# 重启服务
systemctl restart browser_collector

# 停止服务
systemctl stop browser_collector
```

---

## 注意事项

1. **只能内部调用**：8005 端口未对外开放，只能在服务器内部通过 `127.0.0.1:8005` 访问
2. **登录状态**：小红书登录状态存储在 `/www/browser_collector/xiaohongshu_state.json`，登录过期后需重新登录并上传
3. **超时设置**：采集耗时较长（30~120秒），`timeout` 建议设置 180 秒以上
4. **headless 模式**：服务器上以无头模式运行，无浏览器界面
